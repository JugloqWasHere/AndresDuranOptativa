# Creación de HUDs en Unity

Proyecto desarrollado para la asignatura **Optativa**, enfocado en la implementación de los **cuatro tipos de HUD (Heads-Up Display)** vistos en clase, utilizando el motor de desarrollo **Unity**.

## Descripción

Este proyecto consiste en una escena interactiva donde se implementan distintos sistemas de interfaz de usuario (HUDs), aplicando conceptos de diseño visual, programación y experiencia de usuario.

El objetivo principal fue desarrollar un sistema funcional que demostrara el uso práctico de los diferentes tipos de HUD estudiados durante el curso, incorporando elementos interactivos y mecánicas asociadas a cada uno.

## Tipos de HUD implementados

### 1. HUD de Estado (Status HUD)

Permite visualizar información importante del jugador en tiempo real, como:

- Vida
- Energía
- Recursos o estadísticas principales

---

### 2. HUD Informativo

Presenta mensajes e información contextual al usuario durante la interacción con el entorno.

Ejemplos:

- Indicaciones
- Notificaciones
- Información de objetos interactivos

---

### 3. HUD de Navegación

Facilita la orientación del jugador dentro del escenario mediante elementos visuales que indican destinos, direcciones u objetivos.

---

### 4. HUD de Interacción

Permite al usuario interactuar con objetos del entorno mediante mensajes contextuales y acciones específicas.

Ejemplos:

- Presionar una tecla para interactuar
- Activar mecanismos
- Recoger objetos

---
